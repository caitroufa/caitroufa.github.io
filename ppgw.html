Case Study: Effect of a Warming Climate on a Tornado Outbreak Environment Using a Pseudo Pseudo Global Warming Approach

Motivation

What could a tornado outbreak look like in a climate with very little or no anthropogenic warming?
What effect might future warming have on a large tornado outbreak?

Background

In a Pseudo Global Warming (PGW) approach, boundary conditions of a regional climate simulation (RCM) are modified to expected future climate conditions.
For example: $$\Delta T = T(x, y, z)_{\text{future mean}} - T(x, y, z)_{\text{hist mean}}$$ $$T'(x, y, z, t) = T(x, y, z, t) + \Delta T$$
The goal is to compare how environmental conditions for a specific case might differ under different climate conditions.
In this particular case study, I computed a 2°C temperature difference for all locations, thus the extra "pseudo" (what I'm calling PPGW).

Case

3.31.2023 Tornado Outbreak
3rd largest tornado outbreak - 142 tornados with 22 in the LOT region - tied for most tornadoes in LOT in a 24 hr period - 11 EF-3's, 1 EF-4

90 mph straight-line winds

27 fatalities

high bulk shear 60-70kts - MLCAPE 1500+ J/kg

NWS - LOT Severe Storm Summary

Data and Domain

ERA5 surface and pressure level data
3-31, 0Z to 4-1, 21Z
3 hourly data

Domain centered on IL, 548 pts x 563 pts, 3 km grid

Simulation Methods

WRF version 4.3.3 with default physics parameters
1 hour output files

Control run - unmodified ERA5 input data
"Minus" scenario - subtract 2$$\degree$$C from control run metgrid files
"Plus" scenario - add 2$$\degree$$C to control run metgrid files
